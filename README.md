# 🍳 Chefly — Aplicativo de Receitas

## 🇧🇷 Sobre o projeto

Chefly é um aplicativo mobile desenvolvido com React Native que permite explorar e gerenciar receitas de forma simples e intuitiva.

O foco do projeto foi criar uma experiência mobile fluida e organizada. Possui sistema de autenticação pelo google, e por login e senha.

---

## 🇺🇸 About the Project

Chefly is a mobile application built with React Native that allows users to explore and manage recipes in a simple and intuitive way.

The main focus was to create a smooth and well-structured mobile experience. It has authentication via Google, as well as login and password.

---

## 🛠️ Tecnologias | Technologies

- React Native
- TypeScript
- Firebase/Cloud Firestore
- TheMealDB (API)
